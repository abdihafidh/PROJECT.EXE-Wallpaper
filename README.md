# PROJECT.EXE-Wallpaper

Pixelated Creation of Adam wallpaper with PROJECT.EXE overlay (1920x1280)

## Description
A retro-styled desktop wallpaper featuring pixelated art with a PROJECT.EXE themed overlay, inspired by classic Windows 95/98 aesthetics.

## Files
- `wallpaper.png` - Main wallpaper image (1920x1280)
- `generate_wallpaper.py` - Python script to regenerate the wallpaper

## Usage
Run the Python script to generate or customize the wallpaper:
```bash
python generate_wallpaper.py
```

## Requirements
- Python 3.x
- Pillow (PIL)
